1. 프로젝트 환경 세팅
Spring Boot 백엔드, React 프론트엔드 폴더 각각 생성
Git 저장소 초기화, .gitignore 설정 (각 기술별 템플릿 참고)

2. Spring Boot 프로젝트 생성
Spring Initializr 또는 IDE 활용(Gradle/Java 21/Spring Boot 3.3.x)
의존성: Spring Web, Spring Data JPA, Lombok, H2(DB), Springdoc OpenAPI 등

3. React 프로젝트 생성
create-react-app 또는 Vite로 React 앱 생성
폴더 구조 설계, 기본 페이지/라우터 구성

4. 백엔드(REST API) 기본 개발
Entity/Repository/Service/Controller 기본 구조 작성
예시 엔티티(회원, 게시글 등) CRUD API 레이어 완성

5. DB 설정 및 테스트
개발용 H2/PostgreSQL 등 로컬 DB 연결, 데이터 모델링
JPA 동작 확인, 샘플 데이터 입력

6. 프론트엔드(React) 준비 및 API 연동
Axios(fetch) 등으로 REST API 연동
첫 목록 조회/등록 화면 만들기

7. 프론트·백엔드 CORS 및 개발 환경 최적화
Spring Boot CORS, React 개발 서버 프록시 설정

8. 추가 기능 확장
JWT 등 인증 기능(로그인/회원가입)
파일 업로드, 페이지네이션, 검색 등 부가 기능

9. 테스트/문서화
JUnit, Mockito 등으로 백엔드 테스트
springdoc-openapi 등 API 문서 자동화
React 테스트 라이브러리 활용(선택)

10. Docker 등 배포 환경 준비(선택)
Dockerfile/compose로 백엔드·프론트 컨테이너화 학습
